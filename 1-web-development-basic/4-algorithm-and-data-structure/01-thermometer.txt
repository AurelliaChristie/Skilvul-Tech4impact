Sebuah pseudocode tentang cara kerja sebuah thermometer yang akan merubah semua jenis suhu menjadi celcius.

Ada 3 jenis perubahan yang suhu yaitu:

- fahrenheit to celcius = (N - 32) * (5/9)
- kelvin to celcius = (N - 273.15)
- celcius to celcius = N

FUNCTION to_celcius("temp" any number, "type" one of ["Celcius", "Fahrenheit", "Kelvin"]):
	DECLARE "result" AS Number
	IF "type" == "Celcius":
		STORE "temp" to "result"
		RETURN "result"
	ELSE IF "type" == "Fahrenheit":
		CALCULATE ("temp" MINUS 32) TIMES (5 DIVIDE 9) AND STORE TO "result"
		RETURN "result"
	ELSE:
		CALCULATE "temp" MINUS 273.15 AND STORE TO "result"
		RETURN "result"

SHOW "result" // 30
